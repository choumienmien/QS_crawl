# QS_scrapy
 
## QS World University Rankings by Subject 2023
[網路連結](https://www.topuniversities.com/subject-rankings/2023?qs_qp=topnav)

## 說明
 - 動態爬蟲抓取 2023 科目排名
 - 條件:
   *  items_per_page=500 ( 500 筆資料數輸出 )
   *  countries=tw' ( 國家為台灣 )
   *  nid between 3846221 and 3846251 ( 2023 學科代碼區間，每年不一樣 )

## 輸出方式
 -CSV 檔
